# LORIAN STORE
![](/demo.gif)

Fullstack online store with the ability to pay for goods through the "Yookassa" service
## Getting start
### Start back-end:
``` 
cd server-side  
```
``` 
yarn add
```
``` 
yarn start:dev
```

### Start front-end:
``` 
cd client-side 
```
``` 
yarn add
```
``` 
yarn dev
```


## Stack
### Front-end:
- React
- Next JS
- TypeScript
- Redux TLK
- Redux TLK Query

### Back-end:
- Node.js
- NestJS
- PostgreSQL
